# Django-book-store


Store the Book Details:

	Storing the following details associated with it.

		- Book
			* name
			* pages
			* publisher  | (one to many)
			* author 	 | (one to many)
		- Author
			*  name
			*  country
		- Publisher 
			*  name
			*  address_line
			*  state
			*  country
			*  pincode
